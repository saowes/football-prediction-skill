# Football Prediction Skill · 足球比赛赛前深度分析

一个面向 AI Agent（Cursor / Claude）的 **Skill（技能）**，用于生成专业、结构化的**足球比赛赛前深度分析报告**，覆盖战术结构、盘口走势、比分预测、冷门风险与赛后复盘。适用于联赛、杯赛、欧战、洲际赛事、国家队赛事与世界杯。

> ⚠️ **本项目仅用于赛事研究、战术与数据讨论、盘口逻辑探讨，不构成任何投注建议，也不是确定性结论。请理性对待、量力而行。**

---

## 这是什么

这不是一个可运行的程序，而是一套**给大模型使用的提示词技能包（Agent Skill）**。当你把它装进支持 Skill 的 AI 客户端后，向 AI 描述一场比赛，它会扮演"拥有 15 年经验的足球分析师"，按统一模板输出一份逻辑严谨、带表格、带推理、带风险提示的赛前分析报告。

核心理念：**不只给结论，更要给推理过程、对应剧本、盘口逻辑与最大不确定性**，并严格区分「普通胜负 / 竞彩让球 / 亚盘 / 大小球」。

## 核心能力

### 八条核心分析原则（贯穿全文）

1. **不低估非传统强队的防守韧性**（低位防守、反击、定位球、受让盘价值）。
2. **也不低估超级巨星的破局能力**，并区分超级巨星型 / 体系型 / 低效热门三类强队。
3. **严格区分盘口类型**：竞彩让球 ≠ 亚盘 ≠ 欧赔 ≠ 大小球。
4. **市场热度 ≠ 真实稳妥**，识别赔率/水位异常信号。
5. **必须输出风险而非只给结论**，禁止"必胜""稳胆"等绝对化表达。
6. **大赛小组赛末轮：战意重于名气**，提供五类战意模型。
7. **强队胜 ≠ 强队穿盘**，附"强队让球对照矩阵"。
8. **不开胜平负 = 风险集中到深让盘**，需重点防边缘比分。

### 报告结构

- **模块 0（条件触发）**：出线形势与战意核验 —— 仅大赛小组赛**末轮**启用。
- **四大主模块**：
  1. 球队整体画像（排名状态、战术体系、攻防数据、伤停体能）
  2. 关键对位与战术细节（核心对位、中场控制、定位球、教练博弈）
  3. 历史交锋与心理因素（交锋史、心理优势、战意、外部因素）
  4. 盘口走势、比分预测与最终结论（亚盘/欧赔/大小球、竞彩让球判断、剧本推演、最可能比分、冷门风险、结论汇总表）
- **多场分析**：输出"多场风险分级表"（低/中/高/极高）。
- **赛后复盘**：按 `postmatch.md` 输出预测命中统计与模型修正。

## 如何使用

### 方式一：作为 Agent Skill 安装（推荐）

将本项目整个文件夹放入你所用客户端的技能目录，例如：

```text
<项目或用户目录>/.claude/skills/football-prediction-skill/
<项目或用户目录>/.cursor/skills/football-prediction-skill/
```

> `SKILL.md` 中设置了 `disable-model-invocation: true`，即技能**不会被模型自动触发**，需要由你主动调用。当你的提问中出现 `赛前分析`、`盘口`、`亚盘`、`竞彩让球`、`大小球`、`比分预测`、`出线形势`、`战意`、`轮换`、`冷门风险`、`赛后复盘` 等关键词时使用最为合适。

### 方式二：直接作为提示词使用

若你的客户端不支持 Skill 机制，可手动把 `SKILL.md` 的内容作为系统提示词 / 角色设定贴给 AI，并在需要逐节输出时附上 `template.md`（赛后复盘时附上 `postmatch.md`）。

### 使用示例

向 AI 提供尽可能完整的比赛信息后提问，例如：

```text
帮我做一场赛前分析：英超 阿森纳 vs 曼城，主场，竞彩主队让平/半球，大小球 2.5/3，
请重点分析盘口走势、竞彩让球判断和最可能比分。
```

末轮 / 多场 / 复盘示例：

```text
分析世界杯小组赛末轮 A 组两场比赛，给出战意模型、出线形势和多场风险分级表。
```

```text
结合我之前对这场比赛的赛前预测和真实比分 2-1，帮我做一次赛后复盘。
```

**提示数据来源原则**：AI 若能联网会优先使用最新数据（伤停、预计首发、赔率、天气）；若无法联网，会明确标注数据可能滞后，并基于合理假设分析。提供的信息越完整，报告越准确。

## 输出规范

- 关键数据优先用**表格**，关键结论用**加粗**。
- 必须给出推理过程，不只给结论。
- 避免绝对化表达，必须区分各类盘口。
- 必须说明最大不确定性。
- 报告以"最终结论汇总表"结尾。
